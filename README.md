这个项目打算用于个人练习，希望借这个项目来提升自身的技术能力，所以会尽量用到一些新技术来充实自己。
在达到**学习**目的的同时，希望这个项目能做到以下几点：
>
1. **高可用性**
2. **高并发**
3. **信息安全**
4. **维护便捷**
5. **访问速度快**

由于都是个人完成，可能不能做到面面俱到，但还是要尽量做到**力之所及**
